# 🏰 Lobby — Lan House Medieval

Projeto desenvolvido para fins acadêmicos, inspirado em uma Lan House temática medieval chamada Lobby.

O site apresenta os planos da lan house, informações de contato, sistema de reservas e formulário de cadastro com interatividade utilizando HTML, CSS e JavaScript.

---

# 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
